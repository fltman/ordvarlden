Hoppsan.

Låt-läge

Välj en låt — varje sjunget ord blir ett landskap, och resan landar på ordet i samma stund som det sjungs.

Transkriberar …

Hör whisper fel? Klicka på ett ord för att rätta det — töm fältet för att ta bort ordet.

♪
dra = titta · scrolla / W S = fart · Enter = nytt ord

Så funkar Ordvärlden

  • Dra med musen för att titta dig omkring.
  • Scrolla eller håll W/S för att ändra farten längs stigen.
  • Skriv ett ord nedan och tryck Enter — ett nytt landskap tecknas fram och läggs till på planeten.
  • Resan mellan två ord låter den ena teckningen sakta bli den andra.
  • Tryck på ♪ för låt-läget: ladda upp en låt och res genom dess ord i takt med sången.
